**Released June 13, 2026** ·
[Download](https://github.com/raydeng83/Sealshot-Release/releases/tag/v0.5.0)

Sealshot 0.5.0 is the latest direct-download release — signed, notarized, and
delivered automatically to existing installs via Sparkle.

## What's new since 0.4.0

- **Smart Redaction** — on-device detection of emails, phone numbers,
  credit-card numbers, and API keys/tokens (AWS, Stripe, GitHub, JWT, Bearer),
  with a review panel to apply redactions. See
  [Smart Redaction](/docs/guide/redaction/).
- **Enhanced Security** — encrypt your captures, search index, undo history,
  and OCR text at rest, with Touch ID unlock, a one-time recovery key, and
  auto-lock. See [Enhanced Security](/docs/guide/security/).
- **Editor** — split Line and Arrow tools with configurable end-caps and
  dashed/dotted styles; a reworked blur tool (Gaussian or Solid, with a brush
  and strength); object transforms (rotate, flip, exact angle); insert images
  on the canvas; a left Info pane; and a brush-sized blur cursor.
- **Library** — rubber-band (marquee) selection in the grid and recent strip;
  delete and restore are now undoable and redoable, surviving relaunch.
- **Performance** — much faster dragging and a snappier autosave in the editor.

## Notes

- Automatic titles, tags, and categories are temporarily disabled while the
  feature is refined; local OCR and full-text search are unaffected.
