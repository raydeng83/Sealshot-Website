---
title: "Workflow: a screenshot library that remembers for you"
description: Capture anything worth keeping — Sealshot makes it searchable, organized, and private.
---

Order confirmations, error messages, hotel bookings, a settings screen you
finally got right, a paragraph you want to quote — screenshots are the
fastest way to keep anything. The problem was always finding them again.
This workflow turns "screenshot it" into a memory that searches itself.

## The walkthrough

1. **Capture without ceremony.** **⌘⇧C** anything worth keeping. Don't
   name it, don't file it — the point of this workflow is that you don't
   have to. Sealshot OCRs every capture on arrival and
   [generates a title, summary, and Smart Keywords](/docs/guide/ai/)
   from what's visible, all on your Mac.

2. **Let search do the remembering.** Weeks later, type what you remember
   — a vendor, an error code, a confirmation number, a phrase.
   [Search reads inside the images](/docs/guide/library/#search), not just
   titles; on Apple Intelligence Macs it even expands your query to
   related terms.

   ![Search matching text inside captures](/manual/library-search.png)

3. **Skim fast.** Arrow through results and tap **Space** for a
   [Quick Look](/docs/guide/library/#quick-look) — no editor round-trip.
   The date and tag facets in the sidebar narrow things down when search
   is too broad.

4. **Curate only what earns it.** Star the captures you return to
   (**Favorites**), and make a
   [collection](/docs/guide/library/#collections-and-favorites) only when
   a real project emerges — *Apartment hunt*, *Tax 2026*, *Onboarding
   notes*. Everything else stays findable without filing.

5. **Keep it private by default.** This is a diary of your screen — treat
   it like one. Everything is indexed and stored
   [on your Mac only](/docs/guide/security/), and with Enhanced Security
   on, the whole library is encrypted behind Touch ID.

:::tip
Old captures purge themselves from the Trash when deleted, but the
library itself keeps everything until you say otherwise — a capture from
months ago is one search away, which is the whole point.
:::
