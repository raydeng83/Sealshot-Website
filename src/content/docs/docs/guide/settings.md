Open Settings from the menu-bar icon. Settings are grouped into sections down
the left side.

![Settings → General](/manual/settings-general.png)
*The General section of Settings.*

## General

- **Appearance** — match the system theme, or force Light or Dark.
- **Capture Defaults** — the default destination (**Clipboard**, **File**, or
  **Both**), the **save location**, and the **filename format** (the app and
  window name are added automatically; a live example shows the result).
- **Scrolling Capture** — turn **auto-scroll** on or off. With it off, you
  scroll and press Return to finish. *(Direct build only.)*
- **Smart Redaction** — scan captures automatically for sensitive text. See
  [Smart Redaction](/docs/guide/redaction/).
- **Trash** — how many days trashed captures are kept before being purged.
- **Startup** — launch Sealshot automatically at login.
- **Updates** — check for updates automatically (update checks are Sealshot's
  only network activity). *(Direct build only.)*

## Shortcuts

Global keyboard shortcuts for each capture command and for opening the editor.
Every command ships with a default (⌘⇧C, ⌘⇧D, ⌘⇧L, ⌘⇧F, ⌘⇧S, ⌘⇧E) that you
can rebind or clear here. See [Keyboard shortcuts](/docs/guide/shortcuts/).

![Settings → Shortcuts](/manual/settings-shortcuts.png)
*Record a shortcut for each command.*

## Permissions

A live status list for the permissions Sealshot needs — **Screen Recording**
(always) and **Accessibility** (for auto-scroll, direct build only) — with a
button to open the right pane of System Settings.

![Settings → Permissions](/manual/settings-permissions.png)
*Check and grant the permissions Sealshot needs.*
